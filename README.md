# Для коректной работы программы надо:
1) Установить библиотеку PyQt5.
2) Поместить все файлы креме "README" вместе с кодом в одну папку

# Возможные проблемы:
1) Если было введены числа 12 или 1 и следующий вопрос совпадает с предидущем, нету картинки и нету поля с кнопкой для ответа, то надо перезапустить программу

# Как добавить свою задачу:
1) Зайти в файл 'Задачи§1.txt'
2) Написать в конце предидущего вопроса знак (*)
3) Написать после знака свой вопрос 
!ВАЖНО! после последнего вопроса в файле не надо ставить знак (*)
4) Зайти в файл 'Ответы§1.txt'
5) Перейти на следующею сироку
6) Написать ответ на задачу
7) Добавить картинку в оснавной файл (в виде: "Задача" + "номер задачи - 1" + ".png")
!ВАЖНО! картинка подходит только в формате ".png"
